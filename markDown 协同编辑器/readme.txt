lib目录下是markdown转html的外部依赖库
Main目录下是项目的各模块封装的包
view目录下是前端的依赖文件
document目录下可以找到说明文档,可供测试使用的markdown文件,
打包jar用到的MANIFEST.MF
根目录下是markUp.jar,markUp.java,markUp.class
markUp主要是提供一个用户调用的接口,启动整个项目。
打包好的markUp.jar可以直接运行。
命令行中输入 java -jar markUp.jar 即可运行。

项目是在windows平台上vscode编辑,windows cmd 中命令行编译的。
建议在windows平台上运行。其他平台未做过测试，可能会出现字符编码，IP获取失败等问题。。
